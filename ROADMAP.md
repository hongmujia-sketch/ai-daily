## 每日更新流程

1. 在 `index.html` 中找到对应 Day 的 `<div class="day-section" id="dayX">` 区域
2. 替换或新增该 Day 的内容（保留外部 CSS class 结构）
3. 更新 `COURSE_DATA` 中对应 Day 的 `status: "upcoming"` 改为 `status: "done"`
4. 运行部署命令：

```bash
cd /Users/hmjwork/Desktop/AI日报
git add -A
git commit -m "Day X: 主题 (日期)"
git push github main
git push github main:gh-pages
```
